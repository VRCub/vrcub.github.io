# 强力牌子PowerSign

### 强力牌子作为一个非常实用的命令功能，皆在加强命令的方便性

---

## 1.1如何创建强力牌子？

当你放置牌子之后，并且按照"强力牌子的写法"填写后会生成强力牌子，之后就可以使用

## 1.2强力牌子的写法
:::info
**强力牌子写法中，如果"第X行"出现空则代表不用写，如果"需要前置Mod"出现空则。代表这个牌子不需要依赖模组**
:::
